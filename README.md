# AI Quarterly Report (AI 在工程上的應用介紹)

這是一個關於 AI 在工程應用上的介紹網頁專案。

## 專案簡介
本專案展示了如何利用現代化的 Web 技術來呈現 AI 相關的資訊。

## 技術堆疊
*   HTML5
*   Tailwind CSS (via CDN)
*   React (via ESM)
*   Lucide React (Icons)
*   Google Fonts (Noto Sans TC)

## 專案結構
*   `index.html`: 主要的 HTML 文件，包含內嵌的與 React 邏輯。
*   `index.tsx`: (如果有外部引用的話) 

## 如何執行
由於本專案主要使用 CDN 引入依賴，可以直接使用任何靜態 Web Server 或是 VS Code 的 Live Server 插件開啟 `index.html` 即可預覽。
